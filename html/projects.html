    This is a dashboard i made, inspired by the frosted glass theme. It is a easy way to find all of your services. It uses the filesystem + a simple format to generate the dashboard. It is quick and easy to set up. I am currently working on porting it to docker.

  • Ezweb

    This is a simple web server I created to easily merge your CSS, HTML, and JavaScript files. It is user-friendly and easy to deploy, allowing you to automatically add a navigation bar to every page and separate your CSS, HTML, and JavaScript. Additionally, it enables instant page previews. I am also planning to add a compile script that will convert all assets on a page into a single HTML file.
